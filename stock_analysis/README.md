I'm doing this personal project to take a chance to learn Convolutional Neural Network (CNN) in PyTorch. Thanks for all the amazing tutorials online; those are more helpful than I thought! <br />
I collected largest food and tech companies' stock data from Yahoo Finance and compute their time-serial returns on different frequencies (e.g. from daily return all the way to yearly return). Starting from there, I use torch.nn to build a neural network to learn about those image/data features that can be helpful to classify good and bad stocks <br />
On top of those, for each batch of my training data, I use SGD algorithm to update parameters on each step, and then use the best parameters to compute test sample accruracy. <br />
